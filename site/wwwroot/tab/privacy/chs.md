---
title: 隐私政策
excerpt: One More Cap 会把字幕文件和播放同步信息留在你的 Mac 上。
---

生效日期：2026-07-02

One More Cap 的设计目标是在你的 Mac 本地工作。它不需要账户、广告标识符、analytics SDK 或 tracking SDK。

## 数据收集

One More Cap 不收集、出售或共享个人数据。

这个 app 不会把你的字幕文件、电影文件、播放历史或字幕文本上传到 One More Cap 的服务器。

## 本地文件

当你选择或拖入字幕文件时，One More Cap 会在你的 Mac 上读取这个由你选择的文件，用来显示字幕。当前支持的字幕格式是 SRT 和 WebVTT。

## 播放同步

当你点击 Sync 时，One More Cap 可能会向 macOS 请求 Automation 权限，用来读取 QuickTime Player 的当前播放位置。这个信息只用于校准字幕时间。

App Store 版本只和 QuickTime Player 同步，不包含 Apple TV Accessibility sync，也不包含 Sparkle 更新检查。

GitHub 版本可能额外包含通过 Accessibility 实现的 Apple TV sync，以及通过 Sparkle 实现的可选更新检查。这些功能不属于 App Store 版本。

## 网络使用

App Store 版本显示字幕和与 QuickTime Player 同步时，不需要 One More Cap 账户，也不需要 One More Cap 服务器。

从网站、GitHub Releases 或支持页面打开的链接由你的浏览器处理。
