One More Cap is a native macOS menu-bar app for the simple case where a movie is open, a subtitle file is available, and you need one more caption layer.

It loads SRT and WebVTT subtitle files, shows them in a floating overlay, and can sync subtitle timing to QuickTime Player. Subtitle files stay on your Mac.

## What it does

- Opens `.srt`, `.vtt`, and `.webvtt` subtitle files.
- Displays subtitles in a floating overlay above the player.
- Reads QuickTime Player's current playback time when you press Sync.
- Lets you adjust subtitle offset and subtitle width from the overlay toolbar.
- Uses macOS caption appearance settings where possible.

## Builds

The App Store build is QuickTime-only and does not include Sparkle updates or Apple TV Accessibility sync.

The GitHub build can also include Sparkle updates and Apple TV sync through Accessibility.
