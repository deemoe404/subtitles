如果你遇到 app 问题，或者想提交反馈和功能请求，请使用项目 issue tracker：

[打开 GitHub issue](https://github.com/deemoe404/onemorecap/issues)

如果问题涉及隐私，请发邮件到 [me@dee.moe](mailto:me@dee.moe)。不要把私密字幕文件、电影文件、个人数据或私密日志贴到公开 issue 里。

## 报告问题前

建议附上这些信息：

- macOS 版本。
- One More Cap 版本。
- 你使用的是 App Store 版本还是 GitHub 版本。
- 字幕文件类型，例如 `.srt`、`.vtt` 或 `.webvtt`。
- QuickTime Player 是否已经打开，并且正在播放你想同步的电影。

不要在公开 issue 里附上私密电影文件或私密字幕文件，除非你明确想公开分享它们。

## QuickTime 权限

如果 Sync 不能读取 QuickTime Player，请打开 macOS System Settings，进入 Privacy & Security，然后进入 Automation，允许 One More Cap 读取 QuickTime Player。

没有这个权限时 app 仍然可用，但 Sync 不能自动从 QuickTime Player 校准时间。

## 支持的字幕文件

One More Cap 当前支持 SRT 和 WebVTT 字幕文件：

- `.srt`
- `.vtt`
- `.webvtt`

ASS/SSA 样式和 OCR 暂时不在当前范围内。

## 隐私

字幕文件会留在你的 Mac 上。详情见[隐私政策](?tab=privacy&lang=chs)。
